# fuse-labs-node-server
Create a node server

## start the app
> nodemon server.js

## config heroku
#### install heroku cli
> brew install heroku/brew/heroku

#### check the instalation
> heroku --help

#### start heroku
> heroku init

#### login
> heroku login

#### add ssh keys
> heroku keys:add

#### check ssh connection
> ssh -v git@heroku.com
